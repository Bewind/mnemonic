Spinners

Font Awesome comes with the feature
to animate icons using the Solid family (.fas)

  1. Home
  2. Components
  3. Spinners

Styles

Spin

To spin the icons, add the class .fa-spin to the <i> tag containing the Font Awesome classes.

                <i 
                class="fas fa-spinner fa-spin"></i>
  • Spinner
    .fa-spinner
  • Circle Notch
    .fa-circle-notch
  • Sync
    .fa-sync
  • Sync Alt
    .fa-sync

Pulse

To spin the icons with the pulsating animation, add the class .fa-pulse to the <i> tag containing the Font Awesome classes.

                <i 
                class="fas fa-spinner fa-pulse"></i>
