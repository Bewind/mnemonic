Switch

A switch is a UI element that has two states
similar to a light switch: ON and OFF.

  1. Home
  2. Components
  3. Switch

Usage

Use .switch on the input[type="checkbox"] tag to use.

Remove any text within the <label> and use the .switch__button class.

Status

On

Off

Sizes

Add the following classes to your <form> tag to set the size.

Small .form--sm

Medium .form--md

Large .form--lg
