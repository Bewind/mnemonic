State

All accordions are collapsed by default.

To set the state to expanded on page load,
use the .accordion-container--expanded class on the container
and add the .is-expanded class to the <a> tag with the .accordion class.

If you need to trigger the accordion as collapsed,
add the .accordion-container--collapsed class on the container
and remove the .is-expanded class from the <a> tags with the .accordion class.
