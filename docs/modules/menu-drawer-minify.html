Minify Drawer

The minified side menu only shows the icons of buttons and links.
Toggle the   MENU  on the left to minify.

Add the class .minify to your <nav> with the .nav__drawer class to enable minifying your sidemenu.

Customizing Width

By default, the width of the minified side menu is 70px.
If you need to change this:

SCSS

On the _menu.scss, search for the .minified class and change its max-width

JS

On the app.js line 79, change the value of main's margin-left to the new width.
