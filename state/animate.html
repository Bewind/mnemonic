Animate

Styling rules for animations can be found on _animate.scss

Usage

Add the .animated class before adding the custom transition class
on the element. Example: .animated .fadeIn

The .animated class contains the duration of all transitions (Default is 0.3s).
All animations use one duration for consistency.
To modify this, edit the $animateDuration token on _tokens.scss.
