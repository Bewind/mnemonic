Drawer

Toggles a menu on the side of the page.
Toggle the   MENU  on the left for demonstration.

  1. Home
  2. Modules
  3. Drawer

Anatomy

Toggle

The link that the user clicks to toggle the side menu should have the class .nav__drawer-toggle

Container

The container that contains the side menu items should have the class .nav__drawer

Code

SCSS

The styling rules for the side menu can be found on _menu.scss

JavaScript

The scripts for the side menu can be found on app.js using the method toggleSideMenu().

Behavior

Default

By default, the side menu is collapsed on page reload. It also overlaps with the page content when expanded.

Expand

To expand the side menu on page load, add the .expanded class on the div or section with the .nav__drawer class.

Push

To push the content when the side menu is expanded, add the .push class on the div or section with the .nav__drawer class.

Minify

Allows you to minify the sidemenu to only show icons
