Dropdown

  1. Home
  2. Modules
  3. Dropdown

Anatomy

Container

Add the class .dropdown on a list item or a container.

Trigger

Add a link <a> with href="#" or <button> with the class .dropdown__button.
This is what will trigger the dropdown content to show.

Content

Add the class .dropdown__content on the <div> next to the .dropdown__button.
This is where you can place the content of the dropdown such as another menu with a list of menu items.

Orientation

Vertical

Add the class .dropdown--vertical on the .dropdown
Hover on the the Vertical menu item above to demonstrate.

Horizontal

Add the class .dropdown--horizontal on the .dropdown
Hover on the the Horizontal menu item above to demonstrate.
