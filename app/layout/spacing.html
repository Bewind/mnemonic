Spacing

Dictates hierarchy and grouping of elements.

General

The token $baseSpace dictates the base value of the spacing. All element spacing follow the base value. When adjusted, all components should be relative to it.

Computation Value Usage
$baseSpace None 40px Can be changed
$wholeSpace $wholeSpace = $baseSpace 40px Paddings and margins
$halfSpace $halfSpace = $wholeSpace / 2 20px Paddings and margins

Helper Classes

In case there's a need to explicitly set the margins or paddings, use these classes.
