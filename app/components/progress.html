Progress Bar

Indicates the progress of an ongoing process
or the status of completion of an object

Values

Set the minimum, maximum, and the current value of a progress bar
by using the following attributes:

Minimum Value aria-valuemin
Maximum Value aria-valuemax
Current aria-valuenow
