Text Input

A text field is a form element containing a label and an input
that accepts strings, numbers, and plain text.

Label Alignment

By default, all labels are left aligned.
You can customize this by adding the following classes to your <form> tag.

Center Align .label--align-center

Right Align .label--align-right

Sizes

Add the following classes to your <form> tag to set the size.

Default

Small .form--sm

Medium .form--md

Large .form--lg

Validation

Add the following classes to the <input> tag to show if the user input passes or fails the validation.

Passed .is-passed


Failed .is-failed

Muted States

Disabled

Readonly

Is selectable and its contents copyable (and its value is submitted);
but you cannot edit the contents
