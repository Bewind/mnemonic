Banner

Takes the width and height of the viewport and adjusts the image size accordingly

  1. Home
  2. Modules
  3. Banner

Anatomy

Banners should be added on the <header> of the page

Container

The <div> container should use the .banner class.

By default, the banner takes the whole height of the viewport minus the height of
the navbar. Its height is computed as calc(100vh - #{$headerHeight})
wherein the $headerHeight is the global token configured on _token.scss.

Banner Image

Within the .banner container, add the <div> with the class
.banner__image where the banner image should be placed.

Banner Header

Next to the .banner__image within the .banner container, add the <div>
with the class .banner__header where the banner header can be placed.
This is optional.

Banner Content

Within the .banner__header container, you can add a <div> with the class
.banner__content where additional text can be placed. This is optional.

Sizes

Banner images can take whole or half of the page's height.
The following classes should be added to the container with the .banner class.

Whole Banner .banner--whole
