  2. Modules

Usage

Container

Add the class .nav__mega-container on the container.

Menu

Add the class .nav__mega on the <ul>.

Image

Add the class .mega--image on the <li> if the section contains an image with label.

List

Add the class .mega--list on the <li> if the section contains a list of links.
