Lazyload

Mnemonic uses LazySizes to lazyload images.

Usage

Info

Disable cache and use network throttling set at
a low internet speed connection to view the transition.

                  
                    <div 
                    class="animated hiding">

                      <picture>


                        <source 
srcset="images/lazyload-sample-1-thumb.jpg"

                        data-srcset="https://picsum.photos/id/293/1000/1000" 

                        media="(max-width: 768px)"

                        class="lazyload">

 

                        <source srcset="images/lazyload-sample-1-thumb.jpg"
 
                        data-srcset="https://picsum.photos/id/293/1000/1000"
 
                        media="(min-width: 769px)"
 
                        class="lazyload">


                        
                        <img src="images/lazyload-sample-1-thumb.jpg"
 
                        data-src="https://picsum.photos/id/293/1000/1000"
 
                        class="lazyload">



                      </picture>

                    </div>

Classes

Add the class .lazyload to the <source> and <image> tags to lazyload.

Add the classes .animated and .hiding to the container of the image.

Attributes

Add the placeholder image on the srcset and src attributes.

Add the actual image to be lazyloaded on the data-srcset and data-src attributes.

Placeholder

The placeholder image should be smaller than the actual image size.

Use a maximum of 100px as the width of square and landscape images, and 100px as the height of portrait images
